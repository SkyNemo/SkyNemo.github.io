简介

nginx（读音：engine x）是可以作为HTTP和反向代理服务器、邮件代理服务器和通用TCP/UDP代理服务器，通常用于高负载的场景下，因为它的稳定性、丰富的模块库、灵活的配置和低系统资源的消耗而闻名

nginx的版本

mainline version：主线版本，版本号通常是奇数，目前最新但是还未经过大量测试的版本

stable version：最新稳定版本，版本号通常为偶数，经过大量测试，企业中经常使用该版本

lagacy version：历史稳定版本，往期的稳定版本
